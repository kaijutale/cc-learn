# cc-changelog-ja.md 更新時の参照ルール

## 参照元

cc-changelog-ja.md を更新する際は、必ず以下の公式 Changelog を参照すること:

- [Claude Code CHANGELOG.md](https://github.com/anthropics/claude-code/blob/main/CHANGELOG.md)

## 更新手順

1. 上記の公式 Changelog を WebFetch または Firecrawl MCP で取得する
2. cc-changelog-ja.md の最新バージョンと比較し、未翻訳のエントリを特定する
3. 新しいエントリを日本語に翻訳し、cc-changelog-ja.md の先頭（最新バージョンの上）に追加する
4. 既存の翻訳スタイル・フォーマットに合わせること
