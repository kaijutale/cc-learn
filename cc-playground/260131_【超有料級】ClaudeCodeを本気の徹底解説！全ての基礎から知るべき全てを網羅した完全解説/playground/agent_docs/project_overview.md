# プロジェクト概要

このプロジェクトは Claude Code の学習用 playground です。

## 目的

- Claude Code の各種機能を試す
- ベストプラクティスを実践的に学ぶ
- カスタムスラッシュコマンドやスキルの実験

## ディレクトリ構成

```
playground/
├── .claude/
│   ├── CLAUDE.md          # プロジェクト固有の指示
│   └── commands/          # カスタムスラッシュコマンド
├── agent_docs/            # Progressive Disclosure用ドキュメント
├── _docs/
│   └── templates/         # 実装ログ保存先
└── try/                   # 実験・試作用
```
