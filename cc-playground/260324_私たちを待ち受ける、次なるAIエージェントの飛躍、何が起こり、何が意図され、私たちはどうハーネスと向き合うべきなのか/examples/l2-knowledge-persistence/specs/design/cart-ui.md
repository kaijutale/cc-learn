# Design Spec: カートUI

## レイアウト

```
┌─────────────────────────────────────────┐
│  🛒 カート (3点)                         │
├─────────────────────────────────────────┤
│ ┌──────┐  商品名A                       │
│ │ img  │  ¥1,200                        │
│ └──────┘  [−] 2 [+]   小計: ¥2,400  🗑  │
│─────────────────────────────────────────│
│ ┌──────┐  商品名B                       │
│ │ img  │  ¥3,500                        │
│ └──────┘  [−] 1 [+]   小計: ¥3,500  🗑  │
├─────────────────────────────────────────┤
│                      合計: ¥5,900       │
│              [ レジに進む ]              │
└─────────────────────────────────────────┘
```

## コンポーネント構成

```
CartPage
  ├── CartHeader         -- "カート (N点)" 表示
  ├── CartItemList
  │     └── CartItem     -- 商品画像・名前・価格・数量操作・削除
  ├── CartSummary        -- 合計金額・チェックアウトボタン
  └── EmptyCartMessage   -- カートが空の場合のみ表示
```

## スタイル規約

- カード背景: white, border-radius: 8px, shadow-sm
- 商品画像: 80x80px, object-fit: cover, border-radius: 4px
- 数量ボタン: 32x32px, border: 1px solid gray-300, hover: gray-100
- 削除アイコン: text-gray-400, hover: text-red-500
- チェックアウトボタン: bg-emerald-600, hover: bg-emerald-700, text-white, w-full

## 禁止事項（収束値を剥がすための制約）

- グラデーション背景を使わない
- カート内に広告バナーを配置しない
- 数量入力にテキストフィールドを使わない（+/- ボタンのみ）
- モーダルで確認ダイアログを出さない（トーストを使う）
