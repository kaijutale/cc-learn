# Spec: ショッピングカート機能

## 概要

ユーザーが商品をカートに追加し、数量変更・削除を行い、チェックアウトに進める機能。

## 機能要件

### カートへの追加
- 商品詳細ページの「カートに追加」ボタンで追加
- 同一商品を再度追加した場合は数量を+1（重複エントリは作らない）
- 在庫数を超える追加はエラーメッセージを表示

### カート表示
- 各アイテム: 商品名・単価・数量・小計
- 合計金額をカート下部に表示
- 空カート時は「カートは空です」メッセージ + おすすめ商品リンク

### 数量変更
- +/- ボタンで変更
- 最小値: 1（0にすると削除確認ダイアログ）
- 最大値: その商品の在庫数

### 削除
- ゴミ箱アイコンで削除
- 削除後に「元に戻す」トースト表示（5秒間）

## APIエンドポイント

| Method | Path | Request Body | Response |
|--------|------|-------------|----------|
| GET | /api/cart | - | { items: CartItem[], total: number } |
| POST | /api/cart/items | { productId, quantity } | { item: CartItem } |
| PATCH | /api/cart/items/:id | { quantity } | { item: CartItem } |
| DELETE | /api/cart/items/:id | - | { success: boolean } |

## 受け入れ基準

- [ ] カートに商品を追加できる
- [ ] 同一商品の重複追加で数量が増える
- [ ] 在庫超過時にエラーが表示される
- [ ] 数量変更が即座に合計金額に反映される
- [ ] 削除後の「元に戻す」が機能する
- [ ] 空カート時に適切なメッセージが表示される
- [ ] ページリロードでカート内容が保持される（セッション永続化）
