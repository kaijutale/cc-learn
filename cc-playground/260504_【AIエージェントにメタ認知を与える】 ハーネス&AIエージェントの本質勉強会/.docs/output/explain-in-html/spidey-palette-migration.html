// PART · 03

移行戦略 — 低 blast radius の実装

本移行の核心は CSS class を 1 行も書き換えなかったこと。変数名 (--copper / --moss / --rust) は v1 の Editorial Dark Academia 由来で 意味と乖離しているが、CSS の全 reference を rename する負担を避けて 値の書換のみで美学を切り替えた。

これは「将来真剣に rename する時のために原則は残しつつ、今は最短ルートで美学変更を完成させる」という意図的な技術的負債。コメントで「--copper = Spidey red と読み替える」と明示することで、未来の自分への申し送りとした。
